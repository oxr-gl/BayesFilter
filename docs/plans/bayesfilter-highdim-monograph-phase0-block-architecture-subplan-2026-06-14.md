# Phase 0 Subplan: High-Dimensional Block Architecture And Import/Export Contracts

**Date:** 2026-06-14  
**Master program:** `docs/plans/bayesfilter-highdim-monograph-rewrite-master-program-2026-06-14.md`

## Purpose

Define the chapter responsibilities and import/export logic for the rewritten
high-dimensional nonlinear filtering block before any chapter-level rewriting
begins.

## Questions to answer

1. What exact book-level question does the block answer?
2. What does each chapter import from earlier monograph chapters?
3. What does each chapter contribute uniquely?
4. What does each chapter export to the next chapter?
5. Which material from the current `ch34`--`ch37` is worth salvaging and which
   should be treated as raw material only?

## Deliverables

- a chapter-role map for `ch33`--`ch37`;
- a dependency map to earlier chapters such as sigma-point filters, validation,
  and HMC target-discipline chapters;
- a statement of what `p47` and `p50` each own as authored source bases.

## Verification

The phase succeeds when the rewritten high-dimensional block can be described as
one coherent sequence with explicit import/export logic rather than as a set of
parallel chapter topics.
