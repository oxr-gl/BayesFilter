from upstream adjoint routing into the already-matched current-step local
Jacobian, rather than from local current-step theta arithmetic?

Comparator: local float64 FilterFlow branch, executed read-only as the
canonical executable comparator for this audit lane.

Primary criterion: reconstruct the observed residual
`[5.302734403676368, -0.1337765252068337]` from differences in upstream adjoints
feeding matched local Jacobians, or report that this adjacent boundary does not
explain the residual.

Artifact: preserve the result in

`experiments/dpf_implementation/reports/outputs/dpf_filterflow_float64_row_173_adjacent_boundary_2026-06-04.json`.

## Expected Failure Modes

- Implementation/instrumentation failure: the runner cannot reconstruct the
  same row/time tensors, cannot obtain full-tape adjoints, or cannot compute the
  local-Jacobian crossing without changing the value path.
- Comparator execution failure: the local float64 FilterFlow subprocess or
  reference-status validation fails.
- JSON/serialization failure: tensors cannot be compactly serialized, the JSON
  output cannot be parsed, or the `--validate-only` path fails.
- Diagnostic failure: scalar/value gates pass but no crossing reconstructs the
  residual and no tested stop-gradient control discriminates the boundary.
- Governance failure: any lane-boundary, CPU-only, Claude-review, or
  non-conclusion control fails.

## What Would Change Our Mind

- If one cross-VJP reconstruction matches the observed residual within
  `2e-4`, promote the corresponding adjoint-routing hypothesis as the next
  implementation/debug target.
- If FilterFlow-local/BayesFilter-upstream reproduces the BayesFilter residual
  while BayesFilter-local/FilterFlow-upstream reproduces the FilterFlow side,
  treat upstream adjoint routing as the source.
- If active-sample topology rows have large per-term deltas but zero increment
  delta, keep sample-topology differences as explanatory only.
- If all local-Jacobian crossings fail to reconstruct the residual after value
  gates pass, reject this adjacent boundary and move the cut-set upstream.
- If a veto diagnostic fires, do not update the hypothesis ranking until the
  veto is repaired or the blocker is recorded.

## Hypotheses To Test

H1. Upstream-adjoint mismatch: BayesFilter and FilterFlow have matched local
current-step Jacobians, but different upstream adjoints into
`post_particles`, `proposed_particles`, `proposal_mean`, `proposal_ll`, or
`unnormalized` reconstruct the observed parameter-gradient residual.

H2. Sample-topology cancellation mismatch: individual active-sample
transition/proposal adjoints differ greatly, but their local increment/core
cancellation matches; the residual is caused by which upstream adjoint reaches
the active sample/proposal topology in the full graph.

H3. Carried-history routing mismatch: the residual is generated by the route
through `pre_current_log_likelihoods`, running ESS/log weights, or the
post-resampling state boundary even though the earlier incoming-history push
was nearly aligned.

H4. Adjacent-boundary not source: upstream-adjoint routing through the matched
local current-step Jacobian does not reconstruct the residual, so the cut-set
must move again.

## Skeptical Pre-Execution Audit

- Stale context: the previous direct-theta result supersedes the older
  "direct current-step theta" hypothesis. This plan treats local current-step
  direct arithmetic as ruled out for frozen tensors.
- Wrong baseline: compare only to local float64 executable FilterFlow.
- Proxy metric risk: scalar agreement and finite gradients are gates, not proof
  of correctness.
- Missing stop conditions: stop on subprocess failure, value mismatch, nonfinite
  gradients, lane contamination, Claude/Codex unresolved major disagreement, or
  JSON validation failure.
- Unfair comparison risk: use same row/time, same seeds, same covariance
  convention, same dtype, same frozen tensors, and same local-Jacobian crossing
  protocol.
- Hidden drift: do not edit production, tests, chapters, highdim, vendored code,
  DSGE/NAWM, or FilterFlow source.
- Artifact fit: planned outputs answer whether the adjacent upstream-adjoint
  boundary explains the residual; they do not claim general gradient
  correctness.

The audit passes for planning because the proposed artifacts target only the
next unresolved boundary and preserve the difference-audit governance.

```bash

```

findings must receive a concise rebuttal in the next Claude prompt. Loop until
`ACCEPT` or max five rounds. If a major blocker remains after round five, block
execution pending human direction.
